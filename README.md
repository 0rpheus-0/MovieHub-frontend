# Frontend проекта **MovieHub**

Клиетн для **MovieHub** 

### Технологии

- React

- Docker

### Предварительные действия

Создать файл .env

Добавить переменную
> REACT_APP_API="http://localhost:8080"
___

### Руководство пользователя (кратко)

#### Для запуска (вне Docker):
> *$* npm start

#### Для остановки:

> *Cntr+C*

Backend подымает Frontend
